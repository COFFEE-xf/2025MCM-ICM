# 2025MCM-ICM
Day8 Summary：
本文是一篇关于建模分析英文填词小游戏的论文，为了其时序预测的功能、预测给定单词在具体日期的测试结果的功能和评估单词难易程度的功能，文中分别应用了ARIMA-GARCH模型、Apriori算法和系统聚类算法来解决。
这三种算法同样是文章的创新点，这里重点说一下第一个模型。ARIMA模型并不是第一次出现在O奖论文中，可以说是最常见的时序预测模型之一，而本文的创新点在于将ARIMA模型与另一种时序预测模型GARCH模型结合在了一起，这使得传统ARIMA模型无法用于具有大规模滞后阶数的情况得以解决。同时，文章在建构ARIMA模型的过程中还应用到了AIC和BIC系数，这使得模型构建过程中的主观性因素最大程度上被消除了，我认为这一点是很加分的，这也是我为什么喜欢熵权法大于层次分析法的原因之一。
文章中当然还有其它亮点，比如我以前关注较少的对模型的解释部分。本文在计算出关键方程的系数之后基本上都对其实际意义加以了解释，这使得就算数学功底不好的读者也得以明白这篇论文的计算结果是符合人类直觉的，我觉得这是增强了论文的可读性和说服力的，在比赛中也可以借鉴这一写作方法。
文中基本没有什么大问题，小的笔误都已在笔记中标记，总之这是一篇很优秀的范文，值得作为C题的标杆再三研读。
